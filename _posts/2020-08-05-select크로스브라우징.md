select 태그

1. 기본 스타일 제거
```css
  select {
    border: 0;
    -webkit-appearance: none; /*ios에 나오는 그림자 제거*/
    -moz-appearance: none; /*ios에 나오는 그림자 제거*/
    padding: 0;
    background-color: transparent;
    outline: none;  
  }

  select::::-ms-expand{
    display: none;
  }
```

2. 옵션 스타일링
```css
  /* 첫번째 옵션 제외하고 옵션에 색깔 넣기*/
  select option:not(:first-of-type){
    color: rgba(44, 44, 44, 0.7);
  }
```
